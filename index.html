Abstract

Designing a database for Turf playground are used to play various sports like football, rugby, tennis, cricket, etc. This sports ground booking website is proposed for booking the turf in an easy and efficient way. It has three modules namely, Admin, Manager and User. Admin can login and can add turf locations, assign manager by creating login credentials for manager, add price details for the particular turf, manages turf and view the details of sports venues booking for all locations. Managers assigned by the Admin are different for different Turf playground locations. Managers will get login credentials from admin.
